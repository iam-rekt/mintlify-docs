Give users the best available swap price across the Robinhood Chain ecosystem without making them leave your site.

The RobinSwap widget adds the hosted swap interface with one script. RobinSwap supplies the interface, wallet connection, quotes, and transaction flow; the host website controls placement, sizing, and presentation colors.

## Install the widget

Add a container where the widget should appear, then load the RobinSwap script.

```html
<div id="robinswap-widget"></div>

<script
  src="https://www.robinswap.finance/robinswap-widget.js"
  data-container="#robinswap-widget"
  data-theme="dark"
  data-max-width="520"
  data-height="760">
</script>
```

The container uses the available width while the widget respects `data-max-width`. Place it inside the same responsive column or card used by the rest of the page.

## Customize the surface

Set a complete palette directly on the script:

```html
<script
  src="https://www.robinswap.finance/robinswap-widget.js"
  data-container="#robinswap-widget"
  data-background="#ffffe0"
  data-box-color="#fffef7"
  data-text-color="#10211b"
  data-button-color="#82c900"
  data-radius="18"
  data-max-width="560">
</script>
```

If `data-text-color` is omitted, RobinSwap selects readable light or dark text from the background. The primary button also selects readable text automatically. The RobinSwap name and iframe title remain fixed.

## Available attributes

| Attribute | Purpose |
| --- | --- |
| `data-container` | CSS selector for the host element. Defaults to `#robinswap-widget`. |
| `data-theme` | `light` or `dark`. Omit it to follow the visitor's system theme or the custom background. |
| `data-background` | Six-digit hex color for the widget canvas. |
| `data-box-color` | Six-digit hex color for the network and swap panels. |
| `data-text-color` | Six-digit hex color for primary text, including the header and footer. |
| `data-button-color` | Six-digit hex color for the main Swap or Connect wallet button. |
| `data-from-chain` | Initial source chain ID. The destination remains Robinhood Chain. |
| `data-token-in` | Initial input token contract address. |
| `data-token-out` | Initial output token contract address. |
| `data-amount` | Initial amount in the input token's normal decimal format, such as `25.5`. |
| `data-max-width` | Widget width in pixels, clamped from `320` to `960`. It still contracts on smaller screens. |
| `data-height` | Starting height in pixels, clamped from `560` to `1200`. Open content can grow within that range. |
| `data-radius` | Outer corner radius in pixels, clamped from `0` to `40`. |
| `data-loading` | Set to `eager` to load immediately. Otherwise the iframe uses lazy loading. |

<Info>
  Token values must be valid addresses for the selected source chain. `data-amount` is the amount the user initially pays; users can change it inside the widget.
</Info>

## Wallet connection

The widget contains RobinSwap's wallet setup, so the host website does not install Wagmi or pass a provider into the iframe. **Connect wallet** opens the RobinSwap wallet chooser, including WalletConnect and compatible browser wallets.

Some extensions choose not to expose an injected wallet inside a cross-origin iframe. In that case, use WalletConnect or select **Open RobinSwap** to continue on the full RobinSwap site.
